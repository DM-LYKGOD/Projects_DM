# Lake Eutrophication Model

## Overview

R simulation of lake eutrophication dynamics across multiple nutrient and fishing scenarios. The model tracks phytoplankton, zooplankton, fish, and phosphorus over time.

## Stack

- R
- `deSolve`
- `ggplot2`

## Run

Open `src/lake_eutrophication_model.R` in RStudio or run it with `Rscript`.
