# Renewable Energy Econometrics

## Overview

Python regression workflow examining the relationship between renewable electricity capacity share and GDP while controlling for capital and labour inputs.

## Stack

- Python
- `pandas`
- `numpy`
- `statsmodels`

## Data

Place the source spreadsheet at `data/energy_transition_model_data.xlsx` before running the script.

## Run

Run `src/renewable_energy_econometric_model.py` after providing the spreadsheet in the expected `data/` directory.
