# Data Notes

Current data assets for this project:

- `ml_ready_dataset_transformed.xlsx`
  - Present in this folder and used for downstream modeling and clustering.
- The original wide-format workbook used to create the panel dataset.
  - Optional source workbook if you want to regenerate the transformed file.

The scripts assume the transformed dataset lives at `data/ml_ready_dataset_transformed.xlsx` unless you override the path on the command line.
